Настройки

Рекомендуется: 8-16 потоков для вашего процессора 12/24. Больше потоков = быстрее сканирование.

Отключение CRC32 значительно ускорит сканирование, но уберет возможность поиска дубликатов

💡 Оптимизации производительности:
  • Параллельное чтение файловой системы
  • Асинхронные операции ввода-вывода
  • Пакетная вставка в базу данных
  • Потоковое вычисление CRC32 для больших файлов (64KB буферы)
  • Оптимизированные настройки SQLite (WAL, кэш 10MB)
  • Индексы базы данных для быстрого поиска
